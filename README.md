# 🚀 ResumeIQ AI – AI-Powered ATS Resume Analyzer

ResumeIQ AI is a full-stack AI-powered ATS (Applicant Tracking System) Resume Analyzer that helps students and job seekers improve their resumes and increase their chances of getting shortlisted for placements and internships.

Built using **Java Spring Boot**, **React.js**, **MySQL**, and **Google Gemini AI**, the application analyzes uploaded resumes, calculates an ATS score, and provides recruiter-style feedback with actionable suggestions.

---

## ✨ Features

* 📄 Upload resumes in PDF and DOCX formats
* 🤖 AI-powered resume analysis using Google Gemini AI
* 📊 ATS score calculation (0–100)
* 📝 Extracts skills, education, experience, projects, and certifications
* 💡 Personalized resume improvement suggestions
* 🔍 Missing keyword detection
* 📈 Skill gap analysis
* 🎯 Job role recommendations
* 🏢 Company recommendations
* ✍️ AI-powered resume rewrite suggestions
* 🎤 Mock interview questions
* 📋 Recruiter-style feedback
* 🔐 Secure authentication
* 📱 Responsive dashboard
* ⚡ RESTful API architecture

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ResumeIQ-AI.git
```

### 2. Navigate to the Project

```bash
cd ResumeIQ-AI
```

### 3. Configure the Backend

* Create a MySQL database.
* Update the database credentials in `application.properties`.
* Add your Google Gemini API key.

### 4. Run the Backend

```bash
mvn spring-boot:run
```

### 5. Run the Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🎯 How It Works

1. Register or log in.
2. Upload a PDF or DOCX resume.
3. The application extracts resume information.
4. Gemini AI analyzes the resume.
5. An ATS score is generated.
6. Receive AI-powered feedback and recommendations.
